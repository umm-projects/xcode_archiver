# What?

* `xcodebuild` コマンドを用いて iOS 向けビルド成果物のパッケージングを行います。

# Why?

* 毎回 Xcode を開いてビルドするのも大変なので、自動化してみました。

# Install

```shell
$ npm install github:umm/xcode_archiver.git
```

# Usage

* 組み込むだけで、自動で効果を発揮します。
